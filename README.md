# DocumentManager

Instruction for use
  1. Make sure Node.js is installed on your machine
  2. download the zip file
  3. extract the file
  4. open a command promt and navigate to the project directory (the destination location when you unziped the file)
  5. issue the command "npm install" to install all dependencies needed for use. 
  6. issue the command "node index.js" to start the server.
  7. Open a browser and navigate to http://localhost:3000 to test the service.
